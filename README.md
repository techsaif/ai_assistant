# AI Personal Assistant Chatbot

A Python-based AI personal assistant capable of handling voice and text commands, executing system tasks, fetching real-time information, and managing reminders.

## Features
- Voice and text interaction
- Open system applications
- Weather and Wikipedia search
- Reminder management
- Offline text-to-speech

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- OpenWeatherMap API

## How to Run

```bash
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python main.py
